1개월차

Eclipse IDE와 MariaDB(HeidiSQL)의 사용법을 익혔습니다. Dynamic Web Project로 JSP를 생성, HTML과 Java의 기본 문법을 학습했습니다. SQL문법 중 기초적인 DML을 배웠고, 이를 활용해 Java Calendar API를 사용, 다이어리를 제작했습니다.

2개월차

Java, RDBMS 그리고 CSS와 BootStrap의 적용 방법에 대해 학습했습니다. OracleDB로 계층형 쿼리, group by 함수 등에 대해 학습하였고 Github 사용법에 대해 알게 되었습니다. OracleDB와 MariaDB로 JDBC API를 연습하고 MariaDB를 연동하여 사용자 게시판을 제작했습니다.

3개월차

앞서 배운 기술들과 JavaScript(JQuery) 그리고 Github을 활용하여, Model1 방식의 세미 팀프로젝트(쇼핑몰 웹사이트)를 제작했습니다. 계획표를 제작, 담당업무 분배와 일정 관리 등을 통해 협업 방법에 대해 알게 되었습니다.

4개월차

Servlet의 개념에 대해 배워 Model2 기반의 실습을 진행하였습니다. Spring Framework의 사용법을 익히고 Spring Boot, MyBatis를 이용한 MVC 패턴 프로젝트를 구현하는 방법을 실습했습니다.

5개월차

개월차 스프링 부트 기반의 파이널 팀 프로젝트를 진행하였습니다. 요구사항 명세서, ERD 설계,DB설계 등 산출물을 제작하고, 매주 팀원끼리의 코드리뷰 시간을 가졌습니다. 회원가입, 관리자 페이지, 게시판(공지사항, 문의사항, 리뷰), 물품 결재함, 월별매출 기반의 파트 출력을 담당하여 피트니스 그룹웨어 시스템을 구현했습니다.

6개월차

AWS를 활용한 Ubuntu 서버 세팅 및 프로젝트를 배포하였습니다.
